# Executor v3 Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add session affinity (WARM state with Firecracker pause/resume), session-aware pool operator claims, and transparent eBPF-based event logging proxy to the executor.

**Architecture:** The executor gains a WARM state where the VM is paused (Firecracker pause) between requests, preserving in-memory session state. The pool operator's Claim RPC accepts a session_id for routing to warm pods. A cgroup/connect4 eBPF program intercepts pasta's outbound connections and redirects HTTP/HTTPS to a local proxy that logs requests/responses to tenant MongoDB and provides replay caching on resume.

**Tech Stack:** Go, Firecracker Go SDK (pause/resume), cilium/ebpf (eBPF program loading), net/http (forward proxy), containers/common/libnetwork/pasta

**Spec:** `docs/superpowers/specs/2026-03-16-executor-design.md` (v3)

## Phase 3: eBPF Proxy

### Task 7: Write the eBPF connect4 program

**Files:**
- Create: `executor/bpf/connect4.c`

- [ ] **Step 1: Write the BPF C program**

```c
// executor/bpf/connect4.c
#include <linux/bpf.h>
#include <bpf/bpf_helpers.h>
#include <bpf/bpf_endian.h>

#define PROXY_IP   0x0100007f  // 127.0.0.1 in network byte order
#define PROXY_PORT 3128

struct orig_dest {
    __u32 ip;
    __u16 port;
};

struct {
    __uint(type, BPF_MAP_TYPE_HASH);
    __uint(max_entries, 4096);
    __type(key, __u64);            // socket cookie
    __type(value, struct orig_dest);
} orig_dest_map SEC(".maps");

SEC("cgroup/connect4")
int intercept_connect(struct bpf_sock_addr *ctx) {
    __u16 dst_port = bpf_ntohs(ctx->user_port);

    // Only intercept HTTP (80) and HTTPS (443).
    if (dst_port != 80 && dst_port != 443)
        return 1;

    // Save original destination.
    __u64 cookie = bpf_get_socket_cookie(ctx);
    struct orig_dest dest = {
        .ip = ctx->user_ip4,
        .port = ctx->user_port,
    };
    bpf_map_update_elem(&orig_dest_map, &cookie, &dest, BPF_ANY);

    // Rewrite destination to local proxy.
    ctx->user_ip4 = bpf_htonl(PROXY_IP);
    ctx->user_port = bpf_htons(PROXY_PORT);

    return 1;
}

char LICENSE[] SEC("license") = "GPL";
```
